# Agents guidance

## 概览
这是一个面向中文读者整理的 **DGX Spark Playbooks 中文翻译项目**，旨在帮助中文用户更好地理解和使用 NVIDIA DGX Spark。

## 数据来源
- **原始仓库**：https://github.com/NVIDIA/dgx-spark-playbooks
- **官方 Playbooks 站点**：https://build.nvidia.com/spark

我们这个仓库是 [原始仓库](https://github.com/NVIDIA/dgx-spark-playbooks) 的中文翻译版本，文件组织结构跟原始仓库保持一致。

## 翻译规则

### 目录规则
文档 `目录` 的每个节点都对应了正文中的一个章节标题。在处理过程中应该遵循如下规则：
- 保持目录节点名称和章节标题一致
- 目录节点后面的锚点应该能跳转到对应章节
